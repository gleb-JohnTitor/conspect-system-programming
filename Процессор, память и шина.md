
# **процессор память и шина**

Основа любого компьютера служит ***общая шина***, которая представляет из себя кучу маленьких параллельных проводков, такие проводки называют *дорожки*.
*****К шине подключаются:***** CPU (центральных процессор, англ. *CPU - Central processing unit*), RAM (оперативная память, англ. *ram - random acces memory*), контроллеры.
СPU взаимодействует с остальным компонентами ПК через шину, RAM и контроллеры устроенны так, что они игнорируют любую информацию проходящую через шину, кроме той которая адресована конкретно определенному на них адресу. Для этого под шину выделяются дорожки, которые формируют ***шину адресов*** ,а под данные формируются ***шина данных***, а для управление ***шина управление***.

*Таблица (1. Шины, шины и еще раз шины)*

| Шина | Для чего |
| ------ | ------ |
| Шина адресов | Она определяет куда записывать|
|Шина данных| Она определяет что записывать|
|Шина управление| Она передает управляющие сигналы|

Каждая дорожка может находиться в двух состояниях:

*Таблица (2. Нолик и единичка)*
| Состояние | Что означает |
| ------ | ------ |
| *1* | Напряжение |
| *0* | Нулевое напряжение|

Комбинация 1 и 0 - формирует на шине адресов, и все устройства, кроме CPU, включается в работу если шина адресов соответствует их адресу.

RAM состоит из кучу ячеек памяти, которая имеет свой уникальный адрес. С ячейкой можно проделать две операции: записать и прочитать значение.

*Таблица (3. Шины, шины и еще раз шины в деле)*

| Шина  | операция |
| ------ | ------ |
| Шина адресов | Установка адреса на нужной ему ячейки|
| Шина управление | Передача электрического импульса для: записи или чтение|
| Шина данных | В нее передается значение (чтение) или установка нового значение(запись), стоит учесть, что старое содержимое ячейки пропадает|

По шине данных информация передается параллельно: 8 разрядов = 8 дорожек, при операции чтение ячейкам нужно выстроить логическое уравнение для дорожек, а при записи наоборот дорожкам нужно выстроить логическое уравнение для ячеек.

**!ВАЖНО: ЕСЛИ ЭЛЕКТРИЧЕСТВО НЕТ ТО ДАННЫЕ ЯЧЕЕК ТЕРЯЮТСЯ НАВСЕГДА!**

Есть блоки памяти которые представляют собой не оперативную (RAM), а *постоянную (ROM)*. Такая память в свою очередь не поддерживает операцию записи, но с другой стороны, она не зависима от электричества. И стоит учесть что CPU не различает RAM от ROM.


| Характеристика | RAM | ROM |
| ------ | ------ | ------ |
| Энергозависимость| Да |Нет|
| Запись через CPU| Да | Нет|
| Скорость| Быстрая | Медленная|
